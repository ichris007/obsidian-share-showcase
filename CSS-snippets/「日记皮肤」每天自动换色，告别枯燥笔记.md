
Obsidian界面看久了会觉得单调。
于是，我自己做了一套「日记皮肤」。  

配合模板，可以实现**每天自动换色**。
同时也支持**自动适配浅色、深色主题**。 

现在分享给大家，感兴趣的朋友可以玩玩看。

Obsidian玩起来！

## 介绍和使用说明
这是一套7种主题配置，可为日记中的每一天应用专属颜色方案，通过在`cssclasses`中定义即可。
要调整每日颜色，只需修改对应"星期几"类中的颜色值。

比如，今天是星期三
```
---
cssclass: daily wednesday
---
```

这套日记系统是示例库——生产力和人生管理系统[Lifein](https://github.com/ichris007/Obsidian_Lifein)的一部分，感兴趣的可以试试。
